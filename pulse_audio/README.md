# pulse_audio

Pulse Audio example

## About

This is a command line similar to `paman` (PulseAudio Manager)

## What does it do?

- [x] Show server information
- [ ] Show device list
- [x] Show Client list
- [x] Show module list
- [ ] Play audio
- [ ] Record audio

## Things you should now

This example uses Pulse Audio Asynchronous API.
It means it has callbacks everywhere.
